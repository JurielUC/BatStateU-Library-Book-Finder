BatStateU Library Book Finder

Welcome

The Library Book Finder is an application that allows students to find the location of the books they need inside the campus library.

Project Objective

Library Book Finder App aims to help students find books from the library easily. In this application, the student can search or look for books and their location.

Music Player

Library Book Finder App also has a music player where the user can listen to relaxing music while reading.
